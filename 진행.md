## 완성된 페이지 목록

| URL | 설명 |
|-----|------|
| `/` | 홈 (최신글, 공지, 인기글 — DB 연동) |
| `/board/[slug]` | 게시판별 글 목록 + 페이지네이션 |
| `/post/[id]` | 게시글 상세 + 댓글/대댓글 |
| `/write` | 글쓰기 (로그인 필요) |
| `/login` | 로그인 |
| `/register` | 회원가입 |
| `/search` | 검색 (제목+내용) |
| `/my` | 마이페이지 (내 글·댓글·배팅이력) |
| `/betting` | 포인트 배팅 목록 |
| `/betting/[id]` | 경기 배팅 폼 |
| `/admin/matches` | 관리자 경기 관리 |
| `/board/transfer` | 이적시장 (Transfermarkt API) |
| `/board/highlight` | 하이라이트 (ScoreBat API) |

## API 엔드포인트

| Method | URL | 설명 |
|--------|-----|------|
| POST | `/api/register` | 회원가입 |
| POST | `/api/auth/[...nextauth]` | NextAuth |
| POST | `/api/posts` | 글 작성 |
| POST | `/api/posts/[id]/like` | 좋아요 |
| POST | `/api/comments` | 댓글/대댓글 작성 |
| GET | `/api/matches` | 경기 목록 |
| POST | `/api/matches` | 경기 등록 (관리자) |
| POST | `/api/matches/[id]/result` | 결과 확정 + 포인트 지급 (관리자) |
| POST | `/api/predictions` | 배팅 제출 |
